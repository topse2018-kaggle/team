
■本ソフトウェア名
　Logistic Regressionパターン

■動作環境
　Windows10(64bit) Version 1803
　Anaconda 4.5.11 (Jupyter)
　Python 3.6.7

■本プログラムを動かすためにインストールが必要なライブラリ
　なし

■訓練データ、評価データ
　〇入手場所
　　https://www.kaggle.com/c/titanic

　〇ファイル名
　　train.csv（訓練データ）
　　test.csv（評価データ）

■使い方
　(1) kaggle-titanic-lg.ipynb、train.csv、test.csv を同じフォルダに格納する。
　(2) Jupyterを起動し、Jupyter上から「kaggle-titanic-lg.ipynb」を開く。
　(3) "Cell"メニューから「Run All」を選ぶ。
